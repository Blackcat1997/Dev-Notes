* **Seven (Two kinds)**

  * In JavaScript there are two different kinds of data: **Primitive** and **Objects**.

  * Primitive: it's no a object and has no methods

    * Number
    * String
    * Boolean
    * Null: has a value
    * Undefined: no value
    * Symbol: immutable value, means it's a constant

  * Objects: a collection of properties

    ```JavaScript
    var obj = {
      key1: value1,
      key2: value2,
      key3: value3,
      ...
    }
    ```
